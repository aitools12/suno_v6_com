# Suno V6

Suno V6 is an AI music generator for prompts, lyrics, and moods into original songs and background tracks. It helps users turn prompts and reference images into previewable, export-ready music tracks for product visuals, characters, scenes, and creative production.

## What is Suno V6

Suno V6 is a browser-based AI music generation platform for people who need usable browser-based output without starting every object from zero. The public product positioning is centered on prompt-based music generation, lyrics-to-song creation from reference inputs, browser preview, and flexible export workflows.

## How Suno V6 works

1. Enter a text prompt or upload a source image.
2. Generate a song draft and preview it in the browser.
3. Refine the input and export the strongest result for downstream work.
